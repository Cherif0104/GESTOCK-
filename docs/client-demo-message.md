# Message client - Presentation demo GESTOCK

Bonjour,

Merci pour votre interet pour GESTOCK.

Je vous partage une premiere version de demonstration de la plateforme. Il est
important de preciser qu'il s'agit a ce stade d'un **demonstrateur fonctionnel** :
l'objectif est de vous permettre de visualiser l'experience utilisateur, la
logique des modules, les parcours de gestion de stock et le niveau de
personnalisation possible.

Cette version n'est pas encore connectee a vos donnees reelles ni a une base de
donnees de production. Les donnees visibles sont simulees afin de montrer le
fonctionnement attendu.

Vous pouvez deja tester :

- la page de connexion ;
- les comptes de demonstration ;
- la logique multi-organisation ;
- le tableau de bord interactif ;
- le module Articles ;
- le module Catalogue ;
- les premiers ecrans des autres modules ERP.

Pour vous connecter, utilisez un compte de test depuis le menu **Comptes de
test** sur la page de connexion.

Mot de passe de demonstration :

```text
gestock-demo
```

Comptes recommandes :

- `admin@gestock.local` : acces direct au tableau de bord ;
- `stock@gestock.local` : parcours responsable stock ;
- `achats@gestock.local` : parcours multi-organisation ;
- `direction@gestock.local` : parcours direction / securite ;
- `audit@gestock.local` : parcours audit / premiere connexion.

Je vous conseille de tester en priorite :

1. le dashboard ;
2. le module Articles ;
3. le module Catalogue.

Les modules Articles et Catalogue donnent deja une bonne idee du niveau cible :
formulaires configurables, referentiels enrichissables, recherche, filtres,
fiches detaillees, donnees modifiables et workflows metiers simules.

Les autres modules sont presents pour montrer la vision globale de la plateforme,
mais ils ne sont pas encore au meme niveau de profondeur fonctionnelle.

Si l'approche vous convient, la prochaine etape serait de cadrer vos besoins
reels : types d'articles, entrepots, utilisateurs, workflows, niveaux
d'autorisation, documents, rapports et integrations necessaires.

Bien cordialement.
